Formating

Italic

<em>YOUR TEXT</em> = Italic
OR
<i>YOUR TEXT</i>
For example:
(<em>)Maria(</em>) goes to the beach and get's lost.


Bold

<strong>YOUR TEXT</strong> = Bold
OR
<b>YOUR TEXT</b>
For example:
(<strong>)Maria(</strong>) goes to the beach and get's lost.


Colors!

For making texts colored you will need to put
some code inside the <p> or the <h1>


This is easier shown in some code:

<p style="color :green;">Hello World</p> will output:

Hello World


Spacing

Next Line

<br>YOUR TEXT = Next line (No Spacing!!!)
For example:
Maria goes to the beach (<br>)
and get's lost.

<p>YOUR TEXT<p> = Next line (With Spacing!!!)
For example:

(<p>)Thingy(</p>)

Another Thingy

Headers

For headers you can use: <h1>, <h2>, <h3>, <h4>, ...
This goes only only until <h6>, at <h7> it appears as normal text!!!
For example:

(<h1>)Header(</h1>)


(<h2>)Header(<h2>)


Lists

There are 3 types of lists:
  • Ordered lists (noted with <ol>)
  • Unordered lists (noted with: <ul>)
  • Definition lists (noted with: <dl>)

Ordered and Unordered lists

After you start a list (with <ol> or <ul>) you will include every
item between <li> and </li> ("li" stands for list item). After this,
close the list with </ol> or </ul>.

Definition lists

After opening the list (with <dl>) you will put the item you want between
<dt> and </dt>. Giving it a definition is as easy as puting the text
(or whatever else you want to put as definition) in the line imediatly after the item code
and between <dd> and </dd>.

Other things to keep in mind.

Some other things to keep in mind are:
-sections (<section>YOUR CODE</section>)
makes no diference visually, but helps with search engines.
-articles (<article>YOUR CODE</article>)
makes no diference visually, but helps with screen-readears, search engines and for home assistants.
-aside (<aside>YOUR CODE</aside>)
makes no diference visually, but tells screen-readears, search engines and home assistants that that information is not so important.

also
-block quotas (<blockquote>YOUR CODE</blockquote>)
this makes the text appear with more space by the wall, like you see in this lists.
